## Auction API

# Installation
1. Clone the repository git clone git@github.com:ypeskov/go_auction_api.git [folder]
2. cd [folder]
3. Run bash build-and-run.sh (MacOS/Linux). Windows is not supported.
4. The server will be running on localhost:3000
